2. Longest Substring Without Repeating Characters

Use Sliding Window to solve this problem.

Image there is a window cover target string. This window’s length can increase or decrease. The content covered by window is our substring. So, we increase window length if the next character has not shown in window. If the character has shown before, we decrease the window length from left, one by one, at last we will exclude the repeat character and we continue increasing the window until to the end. Finally, we find the longest substring.

4. Longest Palindromic Substring

Firstly, we consider using dynamic programming, cause it obvious can break down into simple sub-problem in a recursive manner.

Suppose we have a matrix called P, P(i,j) = 0 means $S_i$ to $S_j$ is a palindrome, then mathematical formula like this:
$$
P(i,j) = (P(i+1, j-1) \&\& S_i == S_j)
$$

Secondly, we can reverse S, got S’, then the Palindromic substring is S and S’ common subsequence. Of course, we can use KMP algorithm to get the common subsequence.

Finally, there is an algorithm named Manacher’s Algorithm. If S = ‘abaaba’, we change S to T = ‘#a#b#a#a#b#a#’.
